# 📘 Website Informasi MA Batamiyah

## 📖 Deskripsi
Website ini dibuat oleh seorang alumni MA Batamiyah dengan tujuan untuk mempermudah siswa, orang tua, dan masyarakat dalam mencari informasi mengenai sekolah. Website ini diharapkan menjadi sarana informasi yang mudah diakses, cepat, dan akurat.

## 🎯 Tujuan
- Mempermudah akses informasi tentang MA Batamiyah
- Menyediakan informasi sekolah secara jelas dan terstruktur
- Membantu calon siswa dan masyarakat mengenal sekolah lebih dekat

## 🚀 Fitur Utama
Website ini menyediakan beberapa informasi penting, antara lain:
- 🏫 **Identitas Sekolah**
- 🎯 **Visi dan Misi**
- 🏢 **Fasilitas Sekolah**
- 🔍 **Pencarian Informasi**

## 🛠️ Teknologi yang Digunakan
- Frontend: HTML, CSS, JavaScript

## 👤 Pengembang
Website ini dikembangkan oleh:
- Seorang Alumni MA Batamiyah
